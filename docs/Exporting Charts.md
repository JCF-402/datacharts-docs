
# Export Charts

## Save to Vault as PNG/SVG

You can currently save a chart to your vault by right clicking on the chart. 

![](/img/saveVault.png)

Currently SVG is only supported for line charts. **The default path is Attachments/Charts/**. If that path is not in your vault
it will give an error. You can also modify the default path in the plugin settings under Images.
